K-COL Steel Column Design Platform - Corporate Website

## Project Structure

```
homepage/
├── assets/
│   └── images/
```

## Features

- **Responsive Layout**: Fills entire viewport on any screen resolution (4K, 1080p, etc.)
- **Left Sidebar**: Navigation menu, login form, brochure links
- **Main Content Area**: Background image with exchange rates and news
- **Footer**: Contact information and partner logos

## Getting Started

### Development Server

Run a local server to view the page:

```bash
# Using Python
python -m http.server 8080

# Using Node.js
npx serve -p 8080
```

Then open `http://localhost:8080` in your browser.
