설치 방법:
1) 위 링크에서 NotoSansKR-Regular.otf 다운로드 → 필요 시 .ttf 로 이름 변경
2) 이 폴더 (assets/fonts/) 안에 파일 복사
3) 로컬 서버 재시작

확인:
브라우저에서 아래 주소가 열리면 정상입니다.
http://localhost:8080/assets/fonts/NotoSansKR-Regular.ttf
