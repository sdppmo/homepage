# members 서브컬렉션 생성 가이드 (현재 단계)

> 작성일: 2026-01-25  
> 목적: `P1` 문서가 선택된 상태에서 `members` 서브컬렉션 생성

## 📋 현재 상태

스크린샷 기준:
- ✅ `projects` 컬렉션 생성 완료
- ✅ `P1` 문서 생성 완료
- ✅ `P1` 문서 선택됨 (회색 하이라이트)
- ⏳ **다음 단계: `members` 서브컬렉션 생성**

---

## 🎯 다음 단계: `members` 서브컬렉션 생성

### 현재 화면에서 보이는 것

**오른쪽 패널 상단에 두 개의 파란색 버튼이 보입니다:**

1. **"+ 컬렉션 시작"** 버튼 ← **이 버튼을 클릭하세요!**
2. "+ 필드 추가" 버튼

### Step 1: `members` 서브컬렉션 생성

1. **오른쪽 패널 상단의 "+ 컬렉션 시작" 버튼 클릭**
   - 위치: 오른쪽 패널의 맨 위쪽
   - 현재 `P1` 문서 상세 화면이 보이는 오른쪽 패널입니다
   - 이 버튼을 클릭하면 `P1` 문서 아래에 서브컬렉션을 만들 수 있습니다

2. **서브컬렉션 ID 입력**
   - 팝업 창이 나타나면
   - 컬렉션 ID: `members` 입력
   - "다음" 또는 "Next" 클릭

3. **첫 번째 문서 ID 입력 화면**
   - 이 화면에서는 문서 ID를 입력하지 않고
   - "취소" 또는 "Cancel" 클릭
   - (또는 임시 문서 ID를 입력하고 나중에 삭제)

**결과:**
- `projects/P1/members` 서브컬렉션이 생성됩니다
- 왼쪽 패널에서 `projects` → `P1`을 펼치면 `members`가 보입니다

```

    (선택됨)       (선택됨)       [+ 필드 추가]
```

### Step 1 후
```

├── projects   →   members   →   [+ 문서 추가] ← 다음 단계
    └── P1         (서브컬렉션)
        └── members
```

## 🎯 다음 단계: Admin 멤버 문서 생성

`members` 서브컬렉션이 생성되면:

1. **왼쪽 패널에서 `members` 클릭**
   - `projects` → `P1` → `members` 클릭

2. **중간 패널의 "+ 문서 추가" 버튼 클릭**

3. **문서 ID 입력**
   - Admin 사용자의 UID 입력
   - UID 확인: Firebase Console → Authentication → Users

4. **필드 추가**
   - `role` (string): `admin`
   - `allowedProcesses` (array): `[]` (빈 배열)

5. **저장**
